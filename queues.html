Queues

What is a Queue?

  • First in First out data structure
  • items added to the back of the queue, removed from the front

Why Use them?

  • Breadth First Search
    • Example: floodfill
  • Tash Scheduling
    • Example: Queue of work items, worker pull items off of queue for processing
  • Message and Event Management
  • Parsing

STL Queue

It contains the following:

push(T) // add element to back of queue
pop() // remove element from front of queue
T front() // access element at front of queue
T back() // access element at back of queue
bool empty() // return true if empty
size_t size() // return number of elements in queue
  • Use when you don't need a custom queue. It is templatized

Possible implementation Types

  • You can implement it with an Array, Linked list, or even Both!
  • The STL Deque maintains a list of arrays, and STL Queue actually uses Deque by default!

Priority Queues

  • Every item has a priority associated with it
  • An element with higher priority gets dequeued before lower priority items
  • if two elements have the same priority, it's implementation dependent

Use Cases

  • Task Scheduling
  • ER Patients
  • Path finding (Dikjstra's Algo, A*)
  • Graph Optimization
  • Huffman Coding
  • etc.

STL Priority Queue

It contains the following:

void push(T element) // add member   
void pop() // remove top member
const T& top(); // return const reference to top member 
bool empty(); // return true if empty, false otherwise
size_t size(); // return number of elements
Ordering
  • By default it orders items where highest value dequeues first
  • You can specify a comparator in the template args for custom ordering
// Max-heap for integers  
std::priority_queue<int> pq; 
// Min-heap for integers  
std::priority_queue<int, std::vector<int>, std::greater<int>> min_pq;

// compare by priority first, use seqno for tiebreaker.  
template <typename T>
class PQItem {
public:
    T value;
    unsigned priority;
    unsigned seqno;
    PQItem(T value, unsigned priority) 
      : value(value), priority(priority), seqno(nextSeqno()) {}
    // define operator < or use custom comparator
    bool operator<(const PQItem<T>& other) const {
        if(priority == other.priority) { return seqno > other.seqno; } 
        else { return priority < other.priority; }
    }
};
